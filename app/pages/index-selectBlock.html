Usage

Via data attribute or JavaScript, the select block plugin toggles the .selected class on the trigger item.

Data attributes

data option value requirements description
data-toggle "sb" required toggles the select bloks states
data-default any string Optional Set a value what is returned when nothing is selected. When not specified default return value is null
data-value any string required Set a value of current item.

Methods

$('#element').sb('value') Returns a selected option value if not selected reurns default value (see data-default)
