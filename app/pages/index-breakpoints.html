Breakpoints

Kasutatavad breakpointid

veebijuppide juures erivaadete eihitamisel kasutada breakpoint klassifikaatoreid, mis lisatakse ekraani muutumisel html tag'i

  • extra small devices, portrait mode in phones (max width 767px and portrait) screen__xs--portrait
  • extra small devices, landscape mode in phones (max width 767px and landscape) screen__xs--landscape
  • extra small devices, default (max-width: 768px) screen__xs
  • small device ,landscape mode in tablets (min width 768px and landscape) screen__sm--landscape
  • small device ,tablets (min width 768px) screen__sm
  • medium devices, desktops (min width 992px)screen__md
  • large devices, large desktops (min width 1200px) screen__lg

Usage

Kui soovid luua komponenti extra small screeni jaoks ilma portrait ja landscape juhtumita siis void defineerida selle nii:

[class*="screen__xs"] { ... }
