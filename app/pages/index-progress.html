Progress steps

Example

Basic Progress Steps (PS) component just replaces content area in mobile view les than $screen-sm-min. In Desktop it acts like "Tabs" in mobile like "accordion".

data-progress-steps="5" is required fore layout. The value represents the number of steps you got. Content must be inside .progress-content container. The container is placed right after .progress-active step.
